#### 思路一

两个链表相交的点是指：两个指针指向的内容相同，则说明该结点即在A链上也在B链上，进而说明 A 和 B 是相交的。相交是引用相同，那么相交的情况必定是这种

#### 思路二

将链 A 上的节点遍历放入 Set 集合当中

再遍历 B 链表节点，看 Set 集合当中是否有 A 上的节点

第一个相同的节点便是两个链表的交点

#### 思路三

两个链表相交如上图成 Y 字形

设 A 链表长为 x

设 B 链表长为 y

谁长让谁先走 y - x 或者 x - y 步

然后两个链表就会在相同的起点看是一起走

走到相同的节点必定是两个链表的交点
