#### [面试题 02.02. 返回倒数第 k 个节点](https://leetcode-cn.com/problems/kth-node-from-end-of-list-lcci/)

实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。

注意：本题相对原题稍作改动

**示例：**

```
输入： 1->2->3->4->5 和 k = 2
输出： 4
```



**说明：**

给定的 k 保证是有效的。



```c++
class Solution {
public:
    int kthToLast(ListNode* head, int k) {
        ListNode *fast = head;//快指针
        ListNode *slow = head;//慢指针
        while (fast){
            fast = fast->next;
            if(k > 0) k--;
            else slow = slow->next;
        }
        return slow->val;
    }
};
```
