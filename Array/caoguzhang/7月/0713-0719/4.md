题目：

```
给定 S 和 T 两个字符串，当它们分别被输入到空白的文本编辑器后，判断二者是否相等，并返回结果。 # 代表退格字符。

注意：如果对空文本输入退格字符，文本继续为空。

 

示例 1：

输入：S = "ab#c", T = "ad#c"
输出：true
解释：S 和 T 都会变成 “ac”。
示例 2：

输入：S = "ab##", T = "c#d#"
输出：true
解释：S 和 T 都会变成 “”。
示例 3：

输入：S = "a##c", T = "#a#c"
输出：true
解释：S 和 T 都会变成 “c”。
示例 4：

输入：S = "a#c", T = "b"
输出：false
解释：S 会变成 “c”，但 T 仍然是 “b”。
 

提示：

1 <= S.length <= 200
1 <= T.length <= 200
S 和 T 只含有小写字母以及字符 '#'。
 

进阶：

你可以用 O(N) 的时间复杂度和 O(1) 的空间复杂度解决该问题吗？
```

心路历程

```
利用栈的特点，遍历字符串，如果是'#'，删除栈顶，反之则入栈。
```

JAVA

```
public static boolean backspaceCompare(String S, String T) {
        String newS = getNewString(S);
        String newT = getNewString(T);
        return newS.equals(newT);
    }

    public static String getNewString(String oldString) {
        Stack<Character> stack = new Stack();
        for (char c: oldString.toCharArray()) {
            if (c != '#') {
                stack.push(c);
            } else {
                if (!stack.empty()){
                    stack.pop();
                }
            }
        }
        return String.valueOf(stack);
    }
```
